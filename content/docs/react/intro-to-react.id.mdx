# Apa Itu React?

React adalah library JavaScript yang digunakan untuk membangun antarmuka pengguna (user interface) yang dinamis dan interaktif. React dikembangkan dan dipelihara oleh Facebook. Popularitas React meningkat pesat karena **teknik rendering yang efisien**, **komponen yang dapat digunakan kembali**, serta dukungan komunitas yang aktif.

## Konsep Dasar React

### 1. JSX (JavaScript XML)

JSX adalah ekstensi sintaks untuk React yang memungkinkan pengembang menulis elemen HTML di dalam JavaScript. Penggunaan JSX dalam React tidak bersifat wajib, namun dengan JSX, proses pengembangan menjadi lebih mudah dibaca dan ditulis.

```jsx
// Menggunakan JSX
const ele = <h1>Ini contoh JSX</h1>;
 
// Menggunakan JavaScript biasa
const ele = React.createElement('h1', null, 'Ini contoh JSX');
```

### 2. Komponen

Komponen adalah bagian kode yang bersifat independen dan dapat digunakan kembali. Konsep ini mirip dengan fungsi, namun perbedaannya adalah komponen mengembalikan elemen HTML, bukan sekadar nilai seperti angka atau boolean.

Di React, komponen dapat dibuat dengan dua metode: **class component** dan **functional component**. Berikut adalah contoh implementasi keduanya:

#### Contoh Class Component

```jsx
import React from 'react'

class Hello extends React.Component {  
		render() {
		    return <h1>Hello Nekode!</h1>;  
		}
}
```

#### Contoh Functional Component

```jsx
import React from 'react'
 
function Hello(){
	return (
		<h1>Hello Nekode!</h1>
	)
}
```

### 3. State

**State** adalah objek JavaScript yang menyimpan data dan dapat memengaruhi tampilan komponen berdasarkan perubahan data tersebut.

State memungkinkan pengembang membangun UI yang dinamis dan interaktif karena komponen dapat merespons input pengguna, hasil dari API, atau peristiwa lainnya.

### 4. Virtual DOM

React menggunakan Virtual DOM untuk mengoptimalkan pembaruan halaman web. Setiap objek yang ada di DOM asli memiliki salinan di Virtual DOM React. Walaupun struktur Virtual DOM mirip dengan DOM asli, Virtual DOM tidak langsung mengubah tampilan dokumen.

Cara kerja Virtual DOM: setiap komponen di React dapat memiliki **state**, sehingga setiap kali terjadi perubahan **state**, Virtual DOM diperbarui lebih dahulu. Proses ini berlangsung sangat cepat karena tidak melibatkan perubahan langsung pada antarmuka pengguna.

React hanya akan memperbarui bagian yang berubah menggunakan algoritma **diffing**, yang membandingkan Virtual DOM baru dengan versi sebelumnya.

## Sejarah React

React dikembangkan oleh Facebook dan sering digunakan untuk membangun [Single Page Application (SPA)](https://developer.mozilla.org/en-US/docs/Glossary/SPA). React pertama kali dirilis pada tahun 2013 dan sejak itu menjadi salah satu library JavaScript paling populer dalam pengembangan UI.

React dibuat oleh **Jordan Walke**, seorang insinyur perangkat lunak di Facebook, dan awalnya dirilis sebagai proyek open-source di GitHub.

Saat ini, React telah diadopsi secara luas oleh komunitas pengembang. Banyak library serta framework yang dibangun untuk mendukung React. Selain itu, Facebook juga merilis **React Native**, yang memungkinkan pengembang membuat aplikasi mobile native menggunakan React.

## Mengapa Harus Belajar React di 2025?

React adalah salah satu library JavaScript paling populer untuk pengembangan UI dan memiliki komunitas pengembang yang sangat besar. Berikut beberapa alasan mengapa Anda perlu mempelajari React di tahun 2025:

### 1. Persyaratan Belajar yang Sederhana

Untuk mempelajari React, Anda tidak perlu menguasai banyak bahasa pemrograman atau memiliki pengalaman yang luas. Yang diperlukan hanyalah pemahaman yang kuat mengenai **HTML, CSS, dan JavaScript**.

### 2. Komunitas React yang Besar

Karena React merupakan salah satu library JavaScript yang paling banyak digunakan, Anda tidak akan kesulitan mencari bantuan ketika mengalami error atau kendala. Ada banyak komunitas React, baik online maupun offline, yang dapat ditemukan di platform seperti Facebook, X, atau LinkedIn.

### 3. React Dapat Digunakan di Berbagai Platform

Dengan menguasai React, Anda tidak hanya dapat membangun website, tetapi juga dapat membuat **aplikasi mobile dengan React Native** atau bahkan **aplikasi desktop dengan Electron**.

### 4. Peluang Karier dan Gaji yang Menjanjikan

Gaji merupakan salah satu faktor penting dalam memilih teknologi yang ingin dipelajari. Ribuan perusahaan menggunakan React untuk pengembangan aplikasi mereka, sehingga kebutuhan akan pengembang React sangat tinggi.

Menurut [glassdoor.com](http://glassdoor.com), gaji pengembang React pemula di Amerika Serikat berkisar antara **86K hingga 160K USD per tahun**.
