# Apa Itu JSX?

JSX (JavaScript XML) adalah ekstensi sintaks untuk JavaScript yang memungkinkan Anda menulis elemen HTML di dalam JavaScript. Menggunakan JSX dalam React tidak wajib, tetapi akan mempermudah proses pengembangan dalam membaca dan menulis kode.

```jsx
// Menggunakan JSX
const ele = <h1>Ini adalah contoh JSX</h1>;

// Menggunakan JavaScript biasa
const ele = React.createElement('h1', null, 'Ini adalah contoh JSX');
```

## Cara Kerja JSX

JSX bukanlah sintaks yang valid dalam JavaScript atau HTML, sehingga browser tidak dapat membacanya secara langsung. Oleh karena itu, JSX perlu dikompilasi menjadi JavaScript sebelum dapat dijalankan oleh browser.

Proses transpiling ini terjadi selama proses build, sehingga browser tidak mengetahui bahwa situs ini menggunakan JSX. Browser hanya menerima struktur objek yang telah diterjemahkan menggunakan API React.

```jsx
// Ini adalah JSX
const ele = <h1>Ini adalah contoh JSX</h1>;

// Hasil setelah ditranspilasi
const ele = React.createElement('h1', null, 'Ini adalah contoh JSX');
```

## Manfaat Menggunakan JSX

- **Meningkatkan Keterbacaan**: Sintaks JSX yang mirip HTML membuat kode lebih mudah dibaca dan dipahami.
- **Struktur Berbasis Komponen**: JSX cocok dengan arsitektur berbasis komponen di React, memudahkan dalam melihat struktur dan hierarki komponen.
- **Pengalaman Pengembang yang Lebih Baik**: Dengan JSX, pesan kesalahan dan peringatan lebih jelas selama pengembangan, sehingga mempermudah debugging dan pemeliharaan kode.

# Apa Itu Komponen?

Komponen adalah bagian kode yang independen dan dapat digunakan kembali dalam aplikasi React untuk menentukan tampilan UI.

Komponen pada dasarnya adalah fungsi yang mengembalikan UI. Seperti fungsi, komponen juga dapat menerima input yang disebut **props**. Konsep komponen bertujuan untuk membuat kode lebih **modular dan dapat digunakan kembali**. Ingat, **satu komponen hanya bertanggung jawab untuk satu tugas saja**.

## Komponen Kelas

Komponen kelas adalah kelas ES6 yang diperluas dengan `React.Component`. Berikut contoh pembuatan komponen menggunakan komponen kelas:

```jsx
import React from 'react';

class Hello extends React.Component {
  render() {
    return <h1>Halo Nekode!</h1>;
  }
}
```

Menggunakan komponen kelas adalah cara lama dalam React. Kini, lebih disarankan untuk menggunakan **komponen fungsional**.

## Komponen Fungsional

Komponen fungsional lebih sederhana dibandingkan komponen kelas. Komponen ini seperti fungsi JavaScript biasa, tetapi mengembalikan elemen HTML. Dengan komponen fungsional, kita dapat menggunakan [React Hooks](https://react.dev/reference/react/hooks) untuk mengelola **state** dan **lifecycle events**.

Contoh komponen fungsional:

```jsx
import React from 'react';
 
function Hello() {
  return <h1>Halo Nekode!</h1>;
}
```

## Merender Komponen

Seperti disebutkan sebelumnya, komponen mirip dengan fungsi, tetapi alih-alih mengembalikan nilai, komponen mengembalikan elemen UI atau HTML.

```jsx
<Hello />
```

> **Tips:** Nama komponen dalam React ditulis dengan aturan **PascalCase**, di mana setiap kata diawali huruf kapital, contohnya: `HelloWorld`.

# State dan Props

State memainkan peran penting dalam React untuk menciptakan aplikasi web yang interaktif dan dinamis. **State** dalam React adalah objek yang menyimpan nilai atau data, yang menentukan bagaimana komponen berperilaku.

Setiap kali **state** berubah, React akan me-render ulang komponen untuk mencerminkan perubahan data terbaru, sehingga memungkinkan pembuatan UI yang dinamis dan responsif terhadap interaksi pengguna.

## Membuat State

State dapat dibuat dalam komponen kelas atau komponen fungsional.

### State di Komponen Kelas

```jsx
import React from 'react';

class MyComponent extends React.Component {
  constructor(props) {
    super(props);
    this.state = {
      name: 'Nekode',
    };
  }

  render() {
    return <h1>Nama saya {this.state.name}</h1>;
  }
}
```

### State di Komponen Fungsional

```jsx
import React, { useState } from 'react';
 
function MyComponent() {
  const [name, setName] = useState('Nekode');

  return <h1>Halo, nama saya {name}</h1>;
}
```

## Mengupdate State

Untuk membuat aplikasi lebih dinamis, kita bisa memperbarui nilai state berdasarkan interaksi pengguna, misalnya saat tombol diklik.

### Mengupdate State di Komponen Kelas

```jsx
import React from 'react';

class Counter extends React.Component {
  constructor(props) {
    super(props);
    this.state = { count: 0 };
  }

  increment = () => {
    this.setState((prevState) => ({ count: prevState.count + 1 }));
  };

  render() {
    return (
      <div>
        <h1>Hitungan saat ini: {this.state.count}</h1>
        <button onClick={this.increment}>Tambah</button>
      </div>
    );
  }
}
```

### Mengupdate State di Komponen Fungsional

```jsx
import React, { useState } from 'react';
 
function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <h1>Hitungan saat ini: {count}</h1>
      <button onClick={() => setCount((prev) => prev + 1)}>Tambah</button>
    </div>
  );
}
```

# Props

Props digunakan untuk mengirim data dari satu komponen ke komponen lain.

### Menggunakan Props di Komponen Kelas

```jsx
import React, { Component } from 'react';

class Welcome extends Component {
  render() {
    return <h1>Halo, {this.props.name}!</h1>;
  }
}
```

### Menggunakan Props di Komponen Fungsional

```jsx
import React from 'react';

function Welcome({ name }) {
  return <h1>Halo, {name}!</h1>;
}
```

## Menggunakan Props

```jsx
import React from 'react';
import Welcome from './Welcome';

function App() {
  return <Welcome name="Nekode" />;
}
```
