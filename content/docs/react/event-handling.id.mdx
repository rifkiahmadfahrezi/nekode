# Pengenalan

[Event handling](https://developer.mozilla.org/en-US/docs/Web/Events/Event_handlers) adalah salah satu pemahaman paling fundamental yang bisa membuat website kamu menjadi lebih interaktif, dengan mengerti cara melakukan event handling website kamu bisa dibuat mejadi lebih interktif meskipun tidak menggunakan framework/library apapun.

Pada artikel ini kita akan mengenal cara melakukan event handling, disini akan menjelaskan sedikit dasar tentang event handling  di React, termasuk bagaimana caranya:

- Membuat sebuah event
- Passing sebuah event handler sebagai props

# Membuat sebuah event

Untuk membuat sebuah event di react, kamu hanya perlu menulis nama event pada component yang akan kamu beri eventnya, berikut contoh membuat event pada button element.

```jsx
function handler(){
	alert("hello world")
}

<button onClick={handler}>
	click me
</button>
```

# Passing event handler sebagai props 

Passing sebuah event handler sebagai props ke sebuah component adalah parktek yg sering dilakukan di react, dengan melakukan ini lebih reuseable.

Contohya, jika kita membuat **Button** component yang bisa menerima event `onClick` yang akan menunjukan sebuah alert dialog dengan teks `halo my name is ... (titik-titik akan diisi dengan nama dari variable yg dikirimkan ke argumen)`  


```jsx Title="Button.jsx" 
import React from 'react'

export function Button({ handler }){
	return (
		<button onClick={handler} >
			click me
		</button>
	)
}
```

**Penjelasan**

1. Tambahkan props `handler` untuk mengirm sebuah function event handler ke Button element.
    
    `Button({ handler })`
    
2. Lampikan props `handler` ke onClick Event
    
    `<button onClick={handler} >`
    
Dengan mengikuti cara ini kamu bisa membuat button component bisa menerima event melalui props, berikut contoh penggunaanya.

```jsx
import React from 'react'
import Button from './components/Button'

export function App(){
	function handler(name){
		alert(`Halo my name is ${name}`)
	}

	return (
   <div>
         Halo
         <Button handler={()=> handler("Nekode")} />
   </div>
	)
}
```

## Handling banyak event sekaligus

Jika kamu mau meng-handle banyak event dalam sebuah component, kamu bisa menggunakan [rest parameter](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Functions/rest_parameters) ke props, berikut contohnya:

```jsx
import React from 'react'

export function Button({ ...props }){
	return (
		<button {...props}>
			click me
		</button>
	)
}
```

Dengan menggunakan metode ini, kamu tidak hany bisa meng-handle banyak event di sebuah component, tapi kamu juka bisa menambahkan attribute lainnya pada component tersebut seperti id, className dan lainnya, berikut merupakan hasil nya:
