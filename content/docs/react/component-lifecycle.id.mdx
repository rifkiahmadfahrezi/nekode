# Pengantar

Coba bayangkan, sebuah component React itu seperti manusia, mereka terlahir, berevolusi dan pada akhirnya meninggal. React component juga mengikuti sebuah siklus yang mana adalah mounted(lahir), update (evolusi) dan unmount (mati). Proses ini dikenal sebagai **component lifecycle.**

Seperti yang disebutkan sebelumnya, React component memiliki tiga fase siklus, yaitu:

1. Mounting (lahir): Dimana sebuah component dibuat dan ditambahkan ke DOM-nya.
2. Updating (evolusi): Dimana sebuah komponen berubah sebagai respon dari perubahan sebuah state
3. Unmounting (mati):  Dimana component dihapus dari DOM.

# Fase mounting

Fase mounting adalah dimana sebuah component dibuat dan di render ke DOM untuk pertama kalinya. Fase mounting akan terpicu jika:

1. React menambah component ke JSX tree dan melakukan re-render terhadap parent untuk menambahkan component baru
2. Ketikan `root.render(<Component/>)` dipanggil

Pada mase mounting ini akan ada tiga method yang akan dieksekusi, diantaranya: : `componentWillMount`, `render`, dan `componentDidMount`.
/
![mounting.png](/mounting.png)

Urutan eksekusi untuk setiap method dimulai dari atas ke bawah seperti pada gambar diatas. Pada kasus ini `componentWillMount` adalah method yang pertama dieksekusi, lalu pada method `render` akan membuat file JSX kita ddi render ke DOM.

## Fase updating

Pada fase ini kita mencoba melakukan update pada component dengan cara memicu re-rener pada component tersebut, ini biasanya terjadi jika ada perubahan state didalam sebuah component.

![updating.png](/updating.png)

Ini pada dasarnya hampir sama seperti sebelumnya, tapi pada fase updating terdapat method `shouldComponentUpdate`, yang mana akan melakukan pengecekan apakah component ini harus di render ulang (re-render) atau tidak . jika ini mengembalikan nilai `true` maka component tersebut akan melakukan re-render.

# Fase unmounting

Fase ini adalah dimana sebuah component akan dihapus dari DOM. Fase ini akan terpicu jika:

1. Component tersebut sudah tidak ada di JSX tree
2. React melakukan unmount terhadap parent dari component tersebut

pada Fase ini hanya ada satu method yang akan dieksekusi, yaitu method `componentWillUnmount` yang mana akan dipanggil sebelum componen dihapus dari DOM.
