## Menjalankan aplikasi dengan Git dan PM2

Pada contoh kita akan coba clone public repo untuk contoh, jika kau menggunakan private repo kamu bisa menggunakan cara ini untuk [clone private repo ke server](https://dev.to/koddr/quick-how-to-clone-your-private-repository-from-github-to-server-droplet-vds-etc-39jm)

1. Buat direktori baru untuk aplikasi kita lalu clone repository yg ingin kita deploy
    
    ```bash
    mkdir apps
    git clone git@github.com:rifkiahmadfahrezi/nyanboard.git
    # change to your repo url
    ```
    
2. Pindah ke direktori aplikasi lalu jalankan command build aplikasi
    
    ```bash
    cd apps/nyanboard
    pnpm install
    ```
    
    Jika sudah diinstal kamu bisa menjalankannya dengan menjalankan command `pnpm start` lallu coba akses IP dan port yang kamu jalankan contoh `123.13.13.13:3000` . Dengan cara tersebut kamu berhasil men-deploy aplikasi kamu dan bisa diakses melalui IP, tapi cara ini kurang efektif ,supaya lebih kamu bisa menggunakan PM2 untuk menjalankan dan mengelola aplikasi supaya berjalan lebih baik.
    
3. Install PM2
    
    ```bash
    sudo npm install -g pm2
    ```
    
4. Jalankan aplikasi Nextjs kita menggunakan PM2
    
    ```bash
    pm2 start pnpm --"nama-aplikasi" -- start
    ```
    
    Jika sudah berjalan kamu  bisa memantau aplikasi dengan menjalankan command `pm2 status` , dan berikut ini adalah command yg seing digunakan
    
    - Check status: `pm2 status`
    - Tampilkan logs: `pm2 logs` (Tutup dengan `Ctrl + C`)
    - Bersihkan logs: `pm2 flush`
    - Restart app : `pm2 restart nama_aplikasi`
    - Menghentikan aplikasi: `pm2 stop nama_aplikasi`
    - Menghapus aplikasi: `pm2 delete nama_aplikasi`

4. Jalankan command startup

```bash
pm2 startup
```

Command ini membuat PM2 dapat berjalan saat server atau sistem reboot. Dengan kata lain, perintah ini memastikan bahwa proses yang dikelola oleh PM2 akan tetap berjalan meskipun server dimatikan dan dinyalakan kembali.
