## Spesifikasi dan cara pembelian VPS

Untuk pembelian VPS saya beli di Hostinger, dengan paket KVM 1, spesifikasi yang ditawarkan sudah cukup untuk belajar deploy aplikasi sederhana termasuk penggunaanDocker.

Berikut adalah rincian spesifikasi VPS dari pake KVM 1:

- **1** vCPU core
- **4 GB** RAM
- **50 GB** NVMe disk space
- **4 TB** bandwidth
- **Data Center** di Seluruh Dunia
- Sistem operasi **Linux**

### Pembelian VPS

Untuk cara pembelian kamu bisa ikuti step by step ini, pastikan kamu sudah memiliki akun di hostinger:

1. Pilih paket KVM 1, bisa gunakan link yang ada dibawah⬇
<Callout>
   [Beli disini](https://www.hostinger.co.id/cart?product=vps%3Avps_kvm_1&period=12&referral_type=cart_link&REFERRALCODE=N4ERIFKIAHBV&referral_id=0195ee91-c770-735b-b7ac-c759834b67b7)
   Dengan membeli menggunakan link diatas kamu bisa hemat 20% jika membeli minimal dengan durasi 12 bulan, tapi jika ingin membeli untuk 1 bulan saja diskon 20% akan hangus, tetapi kamu bisa coba pakai kode kupon VPS10 untuk medapat diskon 10%
</Callout>
2. Pilih durasi sesuai keinginan, lalu pilih lokasi server dengan latency paling rendah (saya pilih india 77ms)
3. Pilih sistem operasi, karena tujuan kita ingin belajar maka pilih “Plain OS” lalu pilih ubuntu versi terbaru  
4. Kemudian isi alamat dan metode pembayaran yg kamu inginkan
5. Jika pembayaran berhasil kamu akan diarahkan ke halaman baru dimana kamu akan ditanya apakah akan mengaktifkan fitur malware scanner atau tidak (saya pilih tidak)
6. Kemudian kamu diminta membuat password, pastikan passwordnya **mudah kamu ingat,** lalu klik continue maka VPS kamu akan dibuatkan oleh Hostinger
7. Lanjutkan klik **Manage VPS** untuk mengunjungi panel VPS yang baru kita beli di hostinger
